
Installar o pacote globalmente
    $ npm install -g @angular/cli
    $ npm install -g @angular/cli@latest
Desinstalar o angular/cli
    https://cursos.alura.com.br/forum/topico-erro-no-codigo-npm-install-g-angular-cli-10-1-6-175872
Verificar a versão
    $ ng version
    $ ng --version

Criar o projeto
    $ ng new crud-angular
Rodar o projeto
    $ ng serve

Instalar o angular material: https://material.angular.io/guide/getting-started
    $ ng add @angular/material

Criação de módulos: https://angular.io/docs
    $ ng g m courses
    $ ng g m courses --routing

Criar um component
    $ ng g c courses/courses
